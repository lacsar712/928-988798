测试说明

  1. 点击右下角的无障碍按钮（轮椅图标）打开设置面板
  2. 调整字体大小：点击 A- 或 A+ 按钮，范围 80%-150%
  3. 开启高对比度模式：页面变为黑底黄字
  4. 开启护眼模式：页面变为米色背景
  5. 开启 TTS 朗读模式：点击任意文本内容进行朗读
  6. 开启键盘焦点高亮：按 Tab 键查看焦点效果
  7. 点击"保存设置"将偏好保存到 localStorage（无后端时仅本地保存）
  8. 点击"重置默认"恢复所有默认设置

注意：由于当前使用静态服务器，后端 API 暂不可用，偏好仅保存在 localStorage 中。在完整 PHP 环境中会同时保存到服务端数据库。
